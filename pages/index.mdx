# Welcome to eCrowne Documentation

eCrowne provides ESP32 and ESP8266 firmware support for SimHub, enabling sim racers to create custom hardware devices for their racing setup.

## What is eCrowne?

eCrowne develops and maintains firmware that allows ESP32 and ESP8266 microcontrollers to communicate with SimHub, a popular sim racing telemetry and control software.

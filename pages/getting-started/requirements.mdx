# System Requirements

## Software Requirements
- SimHub (latest version recommended)
- Arduino IDE 2.0+ or PlatformIO
- USB Serial Driver (CP210x or CH340 depending on your board)

## Hardware Requirements

### Minimum PC Specifications
- Windows 10 or newer
- USB 2.0 port
- Stable internet connection for initial setup

### Supported Development Boards

#### ESP32
- ESP32 Dev Kit
- ESP32 WROOM
- ESP32 WROVER
- NodeMCU-32S

#### ESP8266
- NodeMCU
- Wemos D1 Mini
- Generic ESP8266 modules

## Development Environment
- Git (optional, for contributing)
- Visual Studio Code (recommended when using PlatformIO)

# Common Issues

## Connection Problems

### Device Not Recognized
**Symptoms:**
- Device doesn't appear in Windows
- "USB Device Not Recognized" error

**Solutions:**
1. Install correct USB drivers
2. Try different USB cable
3. Check Device Manager for errors

### SimHub Connection Failed
**Symptoms:**
- Device shows in Windows but not in SimHub
- Connection drops randomly

**Solutions:**
1. Verify COM port settings
2. Check baudrate configuration
3. Disable power saving on USB ports

## LED Issues

### LEDs Not Responding
**Symptoms:**
- LEDs remain off
- Wrong colors displayed

**Solutions:**
1. Verify power supply
2. Check data pin configuration
3. Validate LED type in firmware
